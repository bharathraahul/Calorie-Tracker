# BiteLog

A modern web application built with Next.js built for people who want to track nutrition without the tedious manual logging

## Features

- **User Authentication**: Secure user registration and login with bcrypt password hashing
- **Database Integration**: PostgreSQL database with Prisma ORM
- **User Management**: Create and manage user accounts with email validation
- **Modern Stack**: Built with Next.js, TypeScript, and Tailwind CSS

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) - React framework for production
- **Database**: PostgreSQL with [Prisma](https://www.prisma.io/) ORM
- **Authentication**: bcryptjs for password hashing
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm, yarn, pnpm, or bun
- PostgreSQL database
- `.env.local` file with `DATABASE_URL`
